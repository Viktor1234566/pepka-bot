# 🐸 Pepka Bot - Telegram Gaming Bot

**Pepka Bot** - це інтерактивний Telegram бот для гри **Harms Combat** з системою кліків для збирання токенів!

## 📱 Основні Можливості

✅ **Система кліків** - натискайте кнопку і збирайте токени  
✅ **Боївка (Combat)** - битися з супротивниками  
✅ **Профіль гравця** - прогрес, рівень, статистика  
✅ **Інвентарь** - зберігання предметів і зброї  
✅ **Рейтинг** - лідерборд гравців  
✅ **Миссії** - квести для завдань  
✅ **Награди** - бонуси за активність  

## 🚀 Початок роботи

### Вимоги
- Python 3.9+
- pip (менеджер пакетів)
- Telegram Bot Token

### Установка

1. **Клонуйте репозиторій**
```bash
git clone https://github.com/Viktor1234566/Pepka-bot.git
cd Pepka-bot
```

2. **Установіть залежності**
```bash
pip install -r requirements.txt
```

3. **Налаштуйте конфіг**
- Відкрийте `config.py`
- Додайте ваш Telegram Bot Token від @BotFather
```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"
```

4. **Запустіть бота**
```bash
python main.py
```

## 🎮 Команди Бота

| Команда | Опис |
|---------|------|
| `/start` | Почати гру і зареєструватись |
| `/profile` | Переглянути свій профіль |
| `/click` | Натиснути кнопку (збирання токенів) |
| `/combat` | Почати боївку |
| `/inventory` | Переглянути інвентарь |
| `/leaderboard` | Переглянути рейтинг |
| `/help` | Отримати допомогу |
| `/stats` | Переглянути статистику |

## 💰 Система Токенів

- **Клік** = +1 токен
- **Комбо-клік** (5 кліків за 5 сек) = +10 бонус токенів
- **Бій** = +5-50 токенів (залежить від супротивника)
- **Миссія** = +20-100 токенів
- **Рівень вгору** = +50 токенів

## ⚔️ Система Боївки

1. Вибір супротивника
2. Боївка в реальному часі
3. Система HP і атак
4. Виграш = награда токенами

## 🐛 Вирішення Проблем

**Бот не відповідає?**
- Перевірте BOT_TOKEN в config.py
- Переконайтесь, що інтернет з'єдн��ння активне
- Перезапустіть бота: `python main.py`

**Помилка БД?**
- Видаліть файл `pepka.db`
- Переустановіть залежності: `pip install --upgrade -r requirements.txt`
